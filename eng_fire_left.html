Fire Detection System
System Components

BB-1439, BB-1444 and after: Temperature-sensing cable for each engine; two red warning annunciators (L ENG FIRE and R ENG FIRE); test switch on copilot's left subpanel; FIRE DET circuit breaker on right side panel (No. 1 dual fed bus).

Prior to BB-1444 (except 1439): Three photoconductive detectors per engine feeding control amplifiers. Left amplifier controls FIRE L ENG light; right amplifier controls FIRE R ENG light.

Indications

When the temperature-sensing cable is activated or light threshold is reached:

  • Appropriate light on warning annunciator panel illuminates
  • With extinguisher system: L or R ENG FIRE PUSH TO EXT switchlights on glareshield illuminate
NOTE: Prior to BB-1444 (except 1439), system may be set off by sunlight entering nacelle openings at appropriate angles.
System Recovery

Assuming integrity of wiring or sensor cable is maintained and fire extinguishes, the warning light will extinguish. Both systems can again detect outbreak of fire.

Fire Extinguishing System
Extinguishing Agent

Fire in engine compartment is smothered by engulfing nacelle with bromotrifluoromethane (CBrF₃) pressurized with dry nitrogen.

System Components

Three spray bars per engine compartment, each supplied by one common fire extinguisher supply cylinder per engine. One squib per bottle incorporates pyrotechnic cartridge that releases entire contents.

Each engine has independent system. Both circuit breakers fed from hot battery bus.

Controls and Indications

Three-lens control indicator on glareshield (when optional extinguisher installed):

  • Red — L (or R) ENG FIRE PUSH TO EXT (indicates detected fire)
  • Amber — D (indicates extinguisher discharged, cylinder empty)
  • Green — OK (confirms circuit continuity during test)
Operation

When red warning indicates fire and confirmed by pilot:

  1. Shut down appropriate (L or R) engine
  2. Depress fire-extinguishing switchlight
  3. This fires appropriate squib, releasing contents through tubing
  4. When bottle discharged, amber D light illuminates
System Testing
Detection System Test

Rotary test switch allows ground or inflight testing:

  • BB-1439, 1444 and after: Switch in DET L or DET R position illuminates corresponding ENG FIRE light
  • Prior to BB-1444 (except 1439): Four-position rotary switch tests each detection section individually

During testing, pilot's and copilot's red MASTER WARNING light flashes. If extinguisher installed, red lenses placarded L ENG FIRE–PUSH TO EXT and R ENGINE FIRE–PUSH TO EXT illuminate.

Extinguishing System Test

For testing extinguishing systems, rotate TEST SWITCH FIRE DET and FIRE EXT through LEFT and RIGHT EXT positions:

  • Amber D light and green OK light should illuminate
  • Indicates bottle charge detector circuitry and squib-firing circuitry operational
  • Confirms squib is in place
Troubleshooting
NO-GO CONDITION: If fire detection annunciators fail to illuminate during testing, malfunction exists in that system. If no response in any position, check circuit breaker.
Limitations & Portable Equipment
System Availability

Detection system operable when electrical power applied to aircraft. Extinguishing system can be discharged at any time since operated from hot battery bus.

LIMITATION: Each engine has self-contained extinguishing system that can be used only once between recharging. Cannot be used to extinguish fire in opposite engine.
Portable Fire Extinguishers

Two portable fire extinguishers installed in aircraft:

  • Cabin: Normally on floor on left side forward of airstair entrance door, aft of rearmost seat
  • Cockpit: Underneath copilot's seat
